RustyCode

AI-powered autonomous development framework built in Rust

48 crates · 1,100+ files · 525K LOC · 10,000+ tests

Install

macOS / Linux

curl -fsSL https://rustycode-ai.github.io/install.sh | sh

Windows (PowerShell)

irm https://rustycode-ai.github.io/install.ps1 | iex

Or download binaries directly. Build from source with cargo build --release.

Quick Start

# Launch interactive TUI $ rustycode tui # Run a one-shot task $ rustycode "fix the authentication bug in src/auth.rs" # Autonomous agent mode $ rustycode agent new "implement user registration endpoint" # Self-update to latest release $ rustycode update

Commands

rustycode "task"Run a single task with the default agent
rustycode tuiLaunch interactive terminal UI with streaming & themes
rustycode agent new "task"Start an autonomous agent session
rustycode agent listList agent sessions
rustycode provider listShow configured LLM providers
rustycode skills listList installed skill plugins
rustycode updateSelf-update to latest stable release
rustycode update --nightlyUpdate to latest nightly build
rustycode update --checkCheck for updates without installing
rustycode benchRun benchmarks (Terminal Bench 2.0 compatible)

Configuration

Create ~/.rustycode/config.toml to set your default provider and model:

# ~/.rustycode/config.toml [provider] default = "anthropic" [provider.anthropic] model = "claude-sonnet-4-20250514" [provider.openai] model = "gpt-4o" [provider.ollama] model = "codellama" base_url = "http://localhost:11434"

Set your API key via environment variable: export ANTHROPIC_API_KEY=sk-...

Run rustycode provider list to see all configured providers.

Features

Autonomous Orchestration

Structured reasoning, quality gates, plan milestones, and multi-step execution strategies.

20+ Built-in Tools

File I/O, bash, grep, glob, LSP, web fetch, notebook editing, git operations, and MCP support.

Skill System

YAML frontmatter skills with path-based activation, brace expansion, and plugin architecture.

Terminal UI

Ratatui-based TUI with streaming output, session management, mouse support, and syntax highlighting.

Sandboxing

macOS Seatbelt and Linux landlock integration for OS-level command isolation.

Team Orchestration

Multi-agent coordination with task profiling, dependencies, and parallel execution.
